Projets

BlueLock – Coffre USB chiffré (Python)

Conception et développement d’un coffre de fichiers sécurisé avec chiffrement AES‑256‑GCM. Gestion sécurisée de la clé maître via Argon2id / Scrypt, fragmentation des fichiers, index chiffré et interface Tkinter multi-plateforme.

Fonctionnalités : création, ouverture, export, suppression de fichiers et rotation de mot de passe.

Obscurate – Obfuscateur Python

Conception et développement d’une bibliothèque Python d’obfuscation automatique visant à protéger le code source contre l’analyse et le reverse engineering. Le système intercepte le premier lancement d’un script, génère une version obfusquée chiffrée et exécute ensuite le code uniquement en mémoire via une machine virtuelle Python dédiée.

Fonctionnalités : obfuscation automatique, chiffrement du code, exécution en mémoire, compatibilité PyInstaller.
